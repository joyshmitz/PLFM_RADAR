## 1. Підсумок Для Керівництва

Було змодельовано один прямокутний мікросмужковий патч-елемент для фазованої решітки сімейства XPA-105 за допомогою OpenEMS FDTD. Результати підтверджують роботу на 10.5 GHz на підкладці Rogers RO4350B і показують, що одиночний елемент придатний для інтеграції у повний масив 8x16.

### Ключові Метрики

| Метрика | Значення | Оцінка |
| --- | --- | --- |
| Резонансна частота | 10.495 GHz | Точно в ціль, 0.05% від 10.5 GHz |
| Втрати на відбиття (`S11`) | -30.6 dB | Відмінне узгодження |
| Вхідний імпеданс | 47.1 + j0.2 Ohm | Майже ідеальне 50 Ohm узгодження |
| Смуга за рівнем -10 dB | 50 MHz (0.48%) | Вузька, що очікувано для тонкої підкладки |
| Пікова директивність | 7.19 dBi | Класичне значення для одиночного патча |
| ККД випромінювання | 56.7% | Адекватно для підкладки 0.102 mm |
| Реалізоване підсилення | 4.72 dBi | Узгоджується з добутком директивності на ККД |

### Ключовий Висновок

Патч-елемент коректно налаштований на 10.5 GHz. Дуже добре узгодження означає мінімальні втрати на відбиття в точці живлення, а одиночна директивність узгоджується з типовою фізикою прямокутного патча. Для 128-елементного масиву це дає очікуване підсилення по нормалі близько 25.8 dBi.

## 2. Параметри Конструкції

### 2.1 Підкладка: Rogers RO4350B

У конструкції використано Rogers RO4350B, високочастотний термореактивний ламінат, який типовий для X-діапазонних радарів і суміжних RF-застосувань. Параметри взято з Qucs-схем, калькулятора патч-антени та документації по стеку плати.

| Параметр | Значення | Джерело |
| --- | --- | --- |
| Діелектрична проникність (`eps_r`) | 3.48 | `patch_antenna.py`, Qucs, datasheet |
| Кут втрат (`tan d`) | 0.0037 | Даташит RO4350B на 10 GHz |
| Товщина підкладки | 0.102 mm | Зображення стеку плати, L1-L2 core |
| Товщина міді | 0.035 mm (1 oz) | Стек плати і визначення `SUBST` у Qucs |

### 2.2 Геометрія Патч-Елемента

| Параметр | Значення | Примітки |
| --- | --- | --- |
| Ширина патча (`W`) | 9.545 mm | Формула Balanis: `c/(2f) * sqrt(2/(eps_r+1))` |
| Довжина патча (`L`) | 7.401 mm | FDTD-підігнано під резонанс 10.5 GHz |
| Тип живлення | Щупове, lumped port | 50 Ohm при `y = 1.49 mm` від центру |
| Земляна площина | 38.1 x 36.0 mm | Близько `lambda/2` запасу за межами патча |
| Крок елементів | 14.285 mm | `lambda/2` на 10.5 GHz |
| Конфігурація масиву | 8x16, 128 елементів | Контекст XPA-105 8x16 |

### 2.3 Помилка В `patch_antenna.py`

У репозиторному калькуляторі розмірів патча є помилка у формулі ефективної діелектричної проникності. У рівнянні Hammerstad у знаменнику має бути ширина патча `W`, але в історичному коді використано `array[1] x h_cu`.

| Параметр | Помилковий варіант з repo | Правильний Hammerstad |
| --- | --- | --- |
| Знаменник формули | `array[1] x h_cu` | `W` |
| `eps_r_eff` | 2.637 | 3.407 |
| Довжина патча | 8.694 mm | 7.641 mm |
| Резонансна частота | ~9.0 GHz | ~10.5 GHz |
| Похибка по частоті | 14% | < 1% |

Якщо залишити цю помилку, патч резонуватиме приблизно на 9.0 GHz замість 10.5 GHz. У підсумках цього звіту використано вже виправлений розрахунок.

## 3. Налаштування Моделювання

### 3.1 Конфігурація FDTD

Антену змодельовано в OpenEMS як повну тривимірну структуру: патч, підкладка, земляна площина і щуп живлення всередині області з поглинальними межами.

| Параметр | Значення |
| --- | --- |
| Збудження | Гаусовий імпульс, `f0 = 10.5 GHz`, `BW = 4 GHz` |
| Частотний прогін | 8.5-12.5 GHz, 801 точка |
| Граничні умови | MUR, 1-й порядок, поглинальні, усі 6 граней |
| Область моделювання | 80 x 80 x 50 mm |
| Кількість комірок | ~360,750 |
| Крок часу | 79.5 fs |
| Максимум кроків | 120,000 |
| Критерій завершення | -50 dB, досягнуто -49 dB |
| Роздільна здатність сітки | `lambda/20` на 12.5 GHz, близько 1.2 mm |
| Комірки по товщині підкладки | 4 комірки на 0.102 mm |

### 3.2 Калібрування Точки Живлення

Положення щупа живлення підібрано ітераційно. Для цієї геометрії крайовий імпеданс становить близько 144 Ohm, а найкраще узгодження з 50 Ohm знайдено на `y = 1.49 mm` від центру патча.

| Ітерація | Зсув живлення | Zin | S11 | Резонанс |
| --- | --- | --- | --- | --- |
| 1, початкова | 2.78 mm | 100.4 + j6.1 Ohm | -9.4 dB | 9.03 GHz |
| 2, виправлення `L` | 2.76 mm | 118.4 - j7.0 Ohm | -7.8 dB | 10.23 GHz |
| 3, ближче | 1.54 mm | 49.3 + j0.5 Ohm | -41.7 dB | 10.17 GHz |
| 4, фінальна | 1.49 mm | 47.1 + j0.2 Ohm | -30.6 dB | 10.50 GHz |

Третя ітерація дала найглибший мінімум по `S11`, але не на цільовій частоті. Фінальна точка на 1.49 mm спеціально обрана так, щоб отримати правильний резонанс на 10.50 GHz.

## 4. Результати

### 4.1 S-Параметри

![Рисунок 4.1: S11](../assets/xpa-105-antenna-report/common/s11-return-loss.png)

Рисунок 4.1. Втрати на відбиття `S11` залежно від частоти.

| Параметр | Значення |
| --- | --- |
| Резонансна частота | 10.495 GHz, ціль 10.5 GHz |
| `S11` у резонансі | -30.63 dB |
| Смуга за рівнем -10 dB | 50 MHz, 10.470-10.520 GHz |
| Відносна смуга | 0.48% |

Це фізично правдоподібна вузька смуга для підкладки 0.102 mm. Звіт окремо підкреслює, що зміна довжини патча на 0.1 mm зсуває резонанс приблизно на 140 MHz, тому вимоги до виробничого допуску тут жорсткі.

### 4.2 Вхідний Імпеданс

![Рисунок 4.2: вхідний імпеданс](../assets/xpa-105-antenna-report/common/input-impedance.png)

Рисунок 4.2. Дійсна та уявна частини вхідного імпедансу залежно від частоти.

| Параметр | Значення |
| --- | --- |
| Zin у резонансі | 47.1 + j0.2 Ohm |
| Пік дійсної частини | ~150 Ohm |
| Нульове перетинання уявної частини | 10.495 GHz |

Оскільки уявна частина переходить через нуль там само, де дійсна наближається до 50 Ohm, резонанс є справжнім, а не випадковим локальним мінімумом. Це також означає, що антену можна напряму вмикати в 50-омний тракт і ADAR1000 без окремої мережі узгодження.

### 4.3 VSWR

![Рисунок 4.3: VSWR](../assets/xpa-105-antenna-report/common/vswr.png)

Рисунок 4.3. Коефіцієнт стоячої хвилі за напругою залежно від частоти.

Мінімальний `VSWR` становить близько `1.06` на 10.495 GHz, що означає дуже якісне узгодження. Межа `VSWR < 2:1` дає ту саму корисну смугу 50 MHz, що й критерій `S11 < -10 dB`.

### 4.4 Діаграма Спрямованості

![Рисунок 4.4: зрізи діаграми спрямованості](../assets/xpa-105-antenna-report/common/radiation-pattern-cartesian.png)

Рисунок 4.4. Зрізи E-площини та H-площини на 10.5 GHz.

| Параметр | Значення |
| --- | --- |
| Пікова директивність | 7.19 dBi |
| HPBW в E-площині | ~75 deg |
| HPBW в H-площині | ~85 deg |
| Відношення вперед/назад | > 15 dB |

Одиночна діаграма задає оболонку, в межах якої масив може електронно сканувати промінь. Ширина пелюстка 75-85 градусів підтримує сканування до приблизно +/-45 градусів без надмірного падіння підсилення.

### 4.5 Полярна Діаграма Спрямованості

![Рисунок 4.5: полярна діаграма](../assets/xpa-105-antenna-report/common/radiation-pattern-polar.png)

Рисунок 4.5. Полярні діаграми для E-площини та H-площини.

Головний пелюсток спрямований по нормалі до поверхні патча. E-площина трохи вужча за H-площину, що відповідає фізиці прямокутного патча, де резонансна довжина і нерезонансна ширина формують різні профілі.

## 5. ККД Та Оцінка Характеристик Масиву

### 5.1 ККД Одиночного Елемента

| Параметр | Значення |
| --- | --- |
| ККД випромінювання | 56.7% |
| Реалізоване підсилення | 4.72 dBi |
| Директивність | 7.19 dBi |
| Перевірка `Gain = D x eta` | `7.19 - 2.47 = 4.72 dBi` |

Основний внесок у втрати дають діелектричні втрати в RO4350B. Звіт також прямо каже, що перехід на товстішу підкладку 0.254 mm міг би підняти ККД приблизно до 75%, але збільшив би профіль масиву.

### 5.2 Оцінка Масиву

| Параметр | Розрахунок | Результат |
| --- | --- | --- |
| Підсилення множника масиву | `10 x log10(128)` | 21.1 dB |
| Директивність масиву | `7.19 + 21.1 dBi` | 28.3 dBi |
| Підсилення масиву по нормалі | `4.72 + 21.1 dBi` | 25.8 dBi |
| Втрати сканування | `~2-3 dB при +/-45 deg` | ~23-24 dBi |
| EIRP при 1 W передавання | `25.8 + 30 dBm` | 55.8 dBm |

У звіті це порівнюється з комерційними X-діапазонними системами з фазованою решіткою від Echodyne та Fortem, і висновок такий: вибрана антенна архітектура є конкурентною за класом підсилення.

## 6. Перевірка Проти Теорії

### 6.1 Швидкі Sanity-Check Перевірки

| Перевірка | Очікування | Виміряно | Статус |
| --- | --- | --- | --- |
| Резонанс біля 10.5 GHz | 10.5 GHz | 10.495 GHz | PASS |
| `S11 < -10 dB` | `< -10 dB` | -30.6 dB | PASS |
| Директивність 6-8 dBi | 6-8 dBi | 7.19 dBi | PASS |
| Смуга < 1% для тонкої підкладки | < 1% | 0.48% | PASS |
| Zin близько 50 Ohm | 50 Ohm | 47.1 Ohm | PASS |
| ККД > 30% | > 30% | 56.7% | PASS |

### 6.2 Порівняння З Аналітичною Теорією

| Параметр | Теорія | Результат FDTD | Похибка |
| --- | --- | --- | --- |
| Резонансна частота | 10.5 GHz | 10.495 GHz | 0.05% |
| Директивність | 6.6-7.5 dBi | 7.19 dBi | У межах очікуваного |
| Ширина патча | 9.545 mm | 9.545 mm | Точно |
| Довжина патча | 7.641 mm | 7.401 mm | На 3.1% коротше |

Звіт трактує цю різницю 3.1% як нормальну: аналітичні формули не враховують паразитну індуктивність щупа, крайові ефекти скінченної земної площини та вплив поглинальних меж.

## 7. Ключові Висновки Та Рекомендації

1. Помилка в `patch_antenna.py` реальна і критична. Якщо її не виправити, конструкція йде приблизно на 14% повз цільову частоту.
2. Підкладка 0.102 mm - це свідомий компроміс: дуже добрий профіль, але лише близько 50 MHz смуги та близько 57% ККД.
3. Оцінка для 128-елементного масиву показує конкурентний клас підсилення для X-діапазонної системи з фазованою решіткою.
4. Чутливість до виробничого допуску висока, тому потрібен RF-grade PCB-процес, а не звичайні припущення рівня FR-4.
5. Коли з'явиться апаратна частина, змодельований `S11` треба напряму звірити з VNA-вимірюванням, бо вузька смуга робить це дуже сильним тестом валідації.
